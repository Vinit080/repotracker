Health Score Calculation

Every repository starts with a perfect score of 100. Points are deducted based on Git hygiene:

  • Uncommitted Changes: -4 pts per dirty file (max -30)
  • Unpushed Commits (Ahead): -6 pts per commit (max -18)
  • Missing Upstream (Behind): -8 pts per commit (max -24)
  • No Remote URL: -10 pts
  • Staleness (Code Rot): -10 pts if >7 days old, -20 pts if >30 days old

Commit often, push regularly, and pull the latest changes to keep your rings perfectly pink!
